# Socket.io Server

Backend server for tinder-like app for picking movies with your friends. App can be used by entire group but it will also work solo. When everyone swipes right on a movie it's a match that will be displayed on everyone's screen.

## ENV's

- TMDB_API_KEY (https://www.themoviedb.org/)
- NODE_ENV (production || development)
- PORT (3000)
